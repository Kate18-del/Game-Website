# 🎮 Game Development Studio

Адаптивный многостраничный сайт студии разработки игр с тёмной темой и оранжевыми акцентами.

## 📄 Страницы

| Страница | Файл | Описание |
|----------|------|----------|
| Главная | `index.html` | Hero, популярные игры, преимущества, проекты, подписка |
| О нас | `about.html` | Информация о студии, преимущества работы с нами |
| Контакты | `contacts.html` | Карта и форма обратной связи |

## 🛠 Технологии

- **HTML5**
- **CSS3** — Flexbox, медиа-запросы, адаптивность, градиенты
- **JavaScript (Vanilla)** — бургер-меню, валидация email
- **Google Fonts** — Poppins

## 🎨 Особенности

- Полностью адаптивный дизайн (десктоп, планшет, мобильные)
- Бургер-меню на мобильных устройствах
- Оригинальный дизайн сохранён без изменений
- Форма обратной связи
- Карта на странице контактов
- Проверка email на валидность
